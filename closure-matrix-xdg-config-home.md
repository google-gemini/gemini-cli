## Deliberate deferrals

- Narrowing strict macOS seatbelt reads from all of `HOME_DIR/.config` to only
  the selected user config directory was not changed in this patch because it
  needs separate macOS validation and may affect broader pre-existing behaviour.
- There is still no high-level integration assertion for the once-per-session
  dual-config warning cadence, although the path-helper/unit coverage is in
  place.
