# Initial Concept

Gemini CLI is an open-source AI agent that brings the power of Gemini directly
into the terminal. It provides lightweight access to Gemini, giving you the most
direct path from your prompt to the model.

# Product Guide

## Vision

To empower developers by integrating advanced AI capabilities directly into
their terminal workflow, making code analysis, generation, and automation
seamless and efficient.

## Target Users

- **Developers:** Seeking AI assistance for coding, debugging, and codebase
  exploration.
- **DevOps Engineers:** Automating terminal-based tasks and workflows.
- **Power Users:** Preferring terminal-first tools for productivity.

## Core Goals

- **Direct Access:** Provide the fastest path from a prompt to a Gemini model
  response.
- **Codebase Awareness:** Enable deep understanding and manipulation of local
  code through file and shell tools.
- **Extensibility:** Allow users to expand capabilities via MCP (Model Context
  Protocol) and custom commands.
- **Developer Experience:** Focus on high performance, minimal friction, and
  idiomatic terminal interactions.

## Key Features

- **Natural Language Coding:** Query, edit, and generate code using simple
  prompts.
- **Built-in Tooling:** Integrated support for file operations, shell execution,
  and web search grounding.
- **MCP Support:** Connect to a vast ecosystem of tools and services.
- **Interactive UI:** A rich React/Ink-based interface designed for the
  terminal.
- **Non-interactive Mode:** Scriptable CLI for integration into automated
  pipelines.
- **Context Management:** Persistent project context via GEMINI.md and session
  checkpointing.
