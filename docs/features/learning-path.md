# Learning Path

Track your progress, earn achievements, and level up as you master Gemini CLI.

## Overview

The Learning Path gamifies your journey with Gemini CLI through:
- **XP System:** Earn experience points for activities
- **10 Levels:** Progress from Novice to Legend
- **33 Achievements:** Unlock rewards for milestones
- **Streak Tracking:** Build daily usage habits
- **Statistics:** Monitor your growth

## Levels

Progress through 10 levels by earning XP:

| Level | Title        | XP Required | Description |
|-------|-------------|-------------|-------------|
| 1     | Novice      | 0           | Just getting started |
| 2     | Beginner    | 100         | Learning the basics |
| 3     | Apprentice  | 250         | Understanding fundamentals |
| 4     | Intermediate| 500         | Building confidence |
| 5     | Advanced    | 1,000       | Mastering core features |
| 6     | Expert      | 2,000       | Deep knowledge and skills |
| 7     | Master      | 3,500       | Exceptional proficiency |
| 8     | Guru        | 5,500       | Teaching others |
| 9     | Sage        | 8,000       | Wisdom and experience |
| 10    | Legend      | 12,000      | Ultimate mastery |

## Earning XP

Gain XP through various activities:

### Achievements
- **Small achievements:** 10-30 XP
- **Medium achievements:** 50-100 XP
- **Large achievements:** 150-300 XP
- **Epic achievements:** 500-1000 XP

### Automatic Tracking
Your activities are tracked automatically:
- Commands executed
- Files created
- Tutorials completed
- Workflows executed
- Suggestions accepted
- Explain mode usage
- Days active
- Consecutive streaks

## Viewing Your Progress

### Dashboard

See your overall progress:
```
/progress dashboard
```

Shows:
- Current level and title
- Total XP and progress to next level
- Achievements unlocked
- Current streak
- Recent achievements

### Achievements

View all achievements:
```
/progress achievements
```

Filter by category:
```
/progress achievements tutorials
/progress achievements workflows
/progress achievements mastery
```

### Statistics

View detailed statistics:
```
/progress stats
```

Shows:
- Activity counts (tasks, tutorials, workflows, etc.)
- Engagement metrics (days active, streak)
- Recent XP gains

### Leaderboard

See your rank and level info:
```
/progress leaderboard
```

Shows:
- Your current rank
- Level and title
- Total XP
- Progress to next level
- Achievement score
- Streak bonus

## Rankings

Your rank is based on achievement completion:
- **Novice** - 0-24% achievements
- **Advanced** - 25-49% achievements
- **Expert** - 50-74% achievements
- **Master** - 75-89% achievements
- **Legend** - 90-100% achievements

## Troubleshooting

**Not earning XP?**
- Ensure you're actively using features
- Check `/progress stats` to verify tracking

**Achievement not unlocking?**
- Verify you've met all requirements
- Some achievements have hidden prerequisites

**Lost progress?**
- Progress is saved automatically to `~/.gemini-cli/learning-path.json`
- Restart CLI if needed
