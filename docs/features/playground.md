# Playground

Practice and learn with 50 interactive coding challenges.

## Overview

The Playground provides hands-on coding challenges to help you master Gemini CLI. Complete challenges to earn points, unlock achievements, and improve your skills.

## Getting Started

**List all challenges:**
```
/playground list
```

**Filter by difficulty:**
```
/playground list beginner
/playground list intermediate
/playground list advanced
/playground list expert
```

**Start a challenge:**
```
/playground start basic-hello
```

## Challenge Difficulties

- **Beginner** (10+ challenges, 10-20 pts): Basic commands and prompts
- **Intermediate** (15 challenges, 25-40 pts): File operations and data processing
- **Advanced** (15 challenges, 50-70 pts): Testing, debugging, API integration
- **Expert** (10 challenges, 100-130 pts): Optimization and architecture

## Using Hints

Each challenge includes helpful hints. Use them wisely as they reduce your score by 5 points each.

```
/playground hint <challenge-id> 0  # First hint
/playground hint <challenge-id> 1  # Second hint
```

## Viewing Solutions

View the complete solution with explanation. **Warning:** This reduces your score by 50%.

```
/playground solution <challenge-id>
```

## Submitting Solutions

```
/playground submit <challenge-id> <your-code>
```

## Daily Challenge

Get a new challenge every day with bonus points:
```
/playground daily
```

Complete daily challenges for +50 bonus points!

## Tracking Progress

**View challenge progress:**
```
/playground progress <challenge-id>
```

**View overall statistics:**
```
/playground stats
```

## Scoring

- Base points per challenge
- First-try bonus: +20 points
- Hint penalty: -5 points per hint
- Solution viewed: -50% of score

## Tips

1. Read the challenge description carefully
2. Start with test cases to understand requirements
3. Use hints strategically
4. Try to solve without viewing solution for maximum points
5. Complete daily challenges for bonus points

## Challenge Categories

- **Basics:** Fundamental commands
- **File Operations:** Working with files
- **Data Processing:** Transform data
- **API Integration:** Working with APIs
- **Testing:** Writing tests
- **Debugging:** Finding bugs
- **Optimization:** Performance
- **Architecture:** System design
