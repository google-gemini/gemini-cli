# TAS_DNA DOI Verification Package (v0.1)

This folder bundles the four artifacts needed for DOI deposit and
downstream verification:

- `zenodo_metadata.json` — Zenodo/DataCite metadata with Tokamak
containment keywords and Genesis Mission linkage.
- `citation.bib` — BibTeX citation including the sovereign license
rider note and artifact hash.
- `LICENSE_RIDER.txt` — Sovereign license rider to accompany CC-BY-SA
4.0.
- `README_OAI_ICF.txt` — OAI-ICF v1.1 manifest describing integrity
proofs and deployment instructions.

Use this directory directly when preparing an upload bundle or compress
it as a single archive for transfer.
